# 天數計算系統｜正式版 V1.5

本版將系統內所有日期欄位統一改為彈性文字輸入，支援西元與民國多種格式；「預計送件日前一年勞資會議區間」保留預計開會日期、星期與區間確認。

## 已完成功能

- 日期區間總天數（起日與迄日皆計入）
- 日期區間換算年／月／日
- 起日加年／月／日推算迄日
- 起日加天數推算迄日（起日為第 1 天）
- 總天數換算年／月／日
- 勞資會議屆期依每 3 個月拆分會議區間
- 同一畫面查看當屆、上一屆、下一屆
- 上一屆與下一屆日期可獨立輸入，不強制連續
- 每次會議可輸入預計開會日期、自動帶出星期
- 自動確認開會日期是否位於該次區間
- 日期區間、起日推算、勞資會議屆期、預計送件日與預計開會日期，皆支援西元／民國與多種分隔格式
- 預計送件日前 1 年的 4 個會議區間，也可輸入預計開會日期並顯示星期與區間確認
- 勞保人數提供區間：申請月份往前第 8 個月至第 3 個月，共 6 個月份
- 跨年度月份自動換算
- 西元／民國日期顯示
- 一鍵複製與最近 20 筆紀錄
- 電腦、平板、手機自適應
- 正式 LOGO、favicon、Apple 主畫面圖示與 Web App manifest

## 所有日期欄位可接受格式

以下格式皆可辨識為同一天：

```text
20260101
2026/01/01
2026-01-01
2026.01.01
1150101
115/01/01
115-01-01
115.01.01
```

也支援不補零的分隔格式，例如 `2026/1/1`、`115/1/1`。

## 計算規則

- 日期區間包含起日與迄日。
- 起日加天數時，起日為第 1 天。
- 天數換算採 1 年＝365 天、1 月＝30 天。
- 起日加年／月／日時採實際日曆日期；月底遇到較短月份會自動調整至該月最後一天。
- 勞資會議區間由各屆屆期起日開始，每 3 個月切分一次，最後一個區間以該屆屆期迄日為準。
- 預計送件日前一年區間為「送件日前 1 年起」至「送件日前 1 日」。
- 勞保人數提供區間為申請月份往前第 8 個月至第 3 個月，共 6 個月份。
