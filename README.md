# 天數計算系統｜正式版 V1.3

本版調整「勞資會議計算區間」，可同時查看當屆、上一屆與下一屆，並在每次會議後輸入預計開會日期，自動帶出星期與區間確認。

## 已完成功能

- 日期區間總天數（起日與迄日皆計入）
- 日期區間換算年／月／日
- 起日加年／月／日推算迄日
- 起日加天數推算迄日（起日為第 1 天）
- 總天數換算年／月／日
- 勞資會議屆期依每 3 個月拆分會議區間
- 同一畫面查看當屆、上一屆、下一屆
- 上一屆與下一屆日期可獨立輸入，不強制連續
- 每次會議可輸入預計開會日期
- 預計開會日期自動帶出星期
- 自動確認開會日期是否位於該次區間
- 不再顯示勞資會議區間天數
- 預計送件日前 1 年的 4 個勞資會議區間
- 勞保人數提供區間：申請月份往前第 8 個月至第 3 個月，共 6 個月份
- 跨年度月份自動換算
- 西元／民國日期顯示
- 一鍵複製與最近 20 筆紀錄
- 電腦、平板、手機自適應
- 正式 LOGO、favicon、Apple 主畫面圖示與 Web App manifest

```bash
npx vercel login
```

```bash
npx vercel --prod
```

## 勞資會議使用方式

1. 輸入當屆屆別、起日及迄日。
2. 視需要輸入上一屆與下一屆的起日、迄日。
3. 按「計算會議區間」。
4. 在各次數後方輸入預計開會日期。
5. 系統會自動帶出星期，並顯示「符合區間」或「超出區間」。

## 計算規則

- 日期區間包含起日與迄日。
- 起日加天數時，起日為第 1 天。
- 天數換算採 1 年＝365 天、1 月＝30 天。
- 起日加年／月／日時採實際日曆日期；月底遇到較短月份會自動調整至該月最後一天。
- 勞資會議區間由各屆屆期起日開始，每 3 個月切分一次，最後一個區間以該屆屆期迄日為準。
- 預計送件日前一年區間為「送件日前 1 年起」至「送件日前 1 日」。
- 勞保人數提供區間為申請月份往前第 8 個月至第 3 個月，共 6 個月份。
