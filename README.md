# 天數計算系統｜正式版 V1.1

已整合正式 LOGO、網站圖示與手機主畫面圖示，可直接部署到 Vercel。

## 已完成

- 日期區間總天數（起日與迄日皆計入）
- 日期區間換算年／月／日
- 起日加年／月／日推算迄日
- 起日加天數推算迄日（起日為第 1 天）
- 總天數換算年／月／日
- 西元／民國日期顯示
- 一鍵複製與最近 20 筆紀錄
- 電腦、平板、手機自適應
- 正式 LOGO、favicon、Apple 主畫面圖示與 Web App manifest

## Vercel 上線

第一次使用請先登入：

```bash
npx vercel login
```

在本資料夾內正式部署：

```bash
npx vercel --prod
```

若專案已經連結過 Vercel，可使用：

```bash
npx vercel --prod --yes
```

## 計算規則

- 日期區間包含起日與迄日。
- 起日加天數時，起日為第 1 天。
- 天數換算採 1 年＝365 天、1 月＝30 天。
- 起日加年／月／日時採實際日曆日期；月底遇到較短月份會自動調整至該月最後一天。
